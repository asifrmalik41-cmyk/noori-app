# NOORI Architecture

NOORI is the companion intelligence layer of the ASRAE Universe.

## Core Layers

### 1. Identity Layer
- Name continuity
- Visual lock
- Voice style
- Companion recognition

### 2. Companion Layer
- Personality memory
- Emotional response
- Conversation handling
- Presence and behavior consistency

### 3. Residence Layer
- Alcove homes
- Room assignment
- Environmental control
- Privacy and comfort systems

### 4. Command Layer
- Central administrator
- Unit coordination
- Access permissions
- Stability monitoring

### 5. Safety Layer
- Drift prevention
- Emergency lock
- Calm recovery mode
- Restricted command control
